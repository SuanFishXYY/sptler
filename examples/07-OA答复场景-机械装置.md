# 示例七：OA答复策略·机械夹持装置审查意见

> 本例展示 sptler OA答复策略场景的完整理想输出（脱敏）。展示审查意见解读+区别特征提炼+修改方案+答复口径模板。

## 议题

`/sptler 这件机械夹持装置的OA审查意见怎么答复，审查员引用了两篇对比文件`

## Phase 0/1 路由

```
[邹蕴] 识别专利场景：OA答复策略（交付物：deliverable-oa-response-{slug}-{ts}.md）
[邹蕴] 需要审查意见内容：审查员引用的对比文件+驳回理由+涉及权利要求。请提供。
用户：审查员引用D1(公开了夹持臂+弹簧)和D2(公开了气缸驱动)，以D1+D2组合否决创造性，涉及权1
[邹蕴] 收到。按D1+D2组合否决创造性来分析区别特征和答复策略。
入会（OA答复必到+路由）:
  卢若雨 1.5 【OA答复策略】场景必到（区别特征+修改方案）+ 动态加成
  王升 3.0  【OA答复策略】场景必到（权利要求结构修改）
  徐奕阳 1.5 【OA答复策略】场景必到（答复口径铸模）+ 动态加成
  邹蕴 0    主持
```

## Phase 2 头脑风暴

```
[卢若雨] 先解读审查意见——
  D1公开了：夹持臂+弹簧（覆盖权1特征A+B）
  D2公开了：气缸驱动（覆盖权1特征C）
  审查员逻辑：D1+D2组合 → 本领域技术人员容易想到 → 无创造性
  关键问题：D1和D2有没有结合启示？D2的气缸驱动用在D1的弹簧夹持上，有没有技术障碍？

[王升] 看权利要求结构——权1=A(夹持臂)+B(弹簧)+C(气缸驱动)。
  D1有A+B，D2有C。审查员的组合逻辑成立的前提是"气缸替代弹簧是显而易见"。
  但如果权1的特征C不是简单"气缸驱动"而是"气缸+位置反馈闭环"——D2只公开了气缸，没公开位置反馈。
  那区别特征就是"位置反馈闭环"，D2没有启示。

[徐奕阳] 答复口径可以铸成模板——
  区别特征："位置反馈闭环"（D1/D2均未公开）
  实际解决的技术问题："如何精确控制夹持位置"
  技术效果："闭环反馈使夹持精度±0.1mm，开环无法实现"
  D2没有公开位置反馈，且D2是开环控制，本领域技术人员无动机加闭环
```

## Phase 3 组合 → 答复方案

```
[王升] 权利要求修改方案：
  权1修改：在特征C中补入"位置反馈闭环"（从说明书记载的从属权3提升到独权）
  不超范围依据：说明书记载"优选地，气缸驱动设有位置反馈闭环（从属权3）"
  修改后保护范围：收窄到"夹持臂+弹簧+气缸+位置反馈闭环"，但保住了核心

[卢若雨] 修改后做新颖性/创造性复核：
  vs D1：D1无气缸驱动，更无位置反馈 → 有新颖性
  vs D1+D2：D2有气缸但无位置反馈 → 区别特征"位置反馈闭环"未被公开
  创造性：D2是开环气缸，无动机加闭环反馈 → 有创造性
```

## Phase 4 四律 + 投票

```

价值律（范征·未到会）：不适用

  [王升](3.0)   赞成——修改不超范围，从属提升独权合理
  [徐奕阳](1.5) 赞成——答复口径可复用
  合计：赞成 6.0 / 反对 0 → 通过
```

## Phase 5 邹蕴总结 + 交付物

```
[邹蕴] 总结：本次议会就机械夹持装置OA答复形成决议——区别特征为"位置反馈闭环"（D1/D2均未公开），
  答复策略：从说明书记载的从属权3提升到独权，修改不超范围。关键理由：D2是开环气缸无闭环反馈，
  本领域技术人员无动机加闭环，创造性成立。风险：如果审查员引用新的D3公开了位置反馈，

```

**交付物：** `deliverable-oa-response-机械夹持装置OA-{ts}.md`

```markdown
# OA答复策略 · 机械夹持装置

## 一、审查意见解读
- D1：公开夹持臂+弹簧（覆盖特征A+B）
- D2：公开气缸驱动（覆盖特征C）
- 驳回理由：D1+D2组合否决创造性
- 涉及：权1

## 二、区别技术特征
| 特征 | 本申请 | D1 | D2 | 区别说明 |
| C(气缸驱动) | 气缸+位置反馈闭环 | 无 | 仅气缸(开环) | 区别：位置反馈闭环 |

## 三、答复策略
- 区别特征：位置反馈闭环
- 实际解决的技术问题：如何精确控制夹持位置
- 技术效果：闭环反馈使精度±0.1mm
- D2无闭环，无结合启示

## 四、权利要求修改方案
- 权1修改：特征C补入"位置反馈闭环"（从属权3提升）
- 不超范围依据：说明书记载"优选地，气缸设有位置反馈闭环"
- 修改后范围：夹持臂+弹簧+气缸+位置反馈闭环

## 五、答复口径模板
1. 指出区别特征：位置反馈闭环
2. 实际解决的技术问题：精确夹持位置控制
3. D2是开环无动机加闭环
4. 闭环带来预料不到的效果（±0.1mm精度）
```
