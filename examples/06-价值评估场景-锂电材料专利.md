# 示例六：专利价值评估·锂电正极材料专利组合

> 本例展示 sptler 专利价值评估场景的完整理想输出（脱敏）。展示五维打分+SABC分级+运营建议。

## 议题

`/sptler 这批锂电正极材料专利做下价值评估分级`

## Phase 0/1 路由

```
[邹蕴] 识别专利场景：专利价值评估（交付物：deliverable-valuation-{slug}-{ts}.md）
[邹蕴] 需要评估对象信息：专利号/组合 + 技术领域 + 现状。请提供，或说"概况"。
用户：5件核心专利，锂电正极材料，已授权，用于动力电池
[邹蕴] 收到。按5件核心专利组合评估。
入会（价值评估必到+路由）:
  范征 3.0  【价值评估】场景必到（价值落点+分级）
  陆诗杰 1.5 【价值评估】场景必到（维度打分+权重）+ 动态加成
  王升 3.0  【价值评估】场景必到（结构+稳定性）
  邹蕴 0    主持
```

## Phase 2 头脑风暴

```
[范征] 这批专利的价值落点在哪？动力电池是高增长市场，5件核心专利如果能覆盖主流正极材料路线，
  战略意义大于单件经济价值。我关心的是：这5件是互补的"组合壁垒"还是分散的"各自为战"？

[陆诗杰] 我定5个维度打分——权利稳定性、技术先进性、经济价值、战略意义、可执行性。
  默认权重按通用场景，但动力电池专利我建议调：经济价值0.30、战略意义0.25、稳定性0.20。
  因为这个行业专利的价值主要在经济和卡位，不是稳定性。

[王升] 从结构看，5件专利的权利要求骨架——3件独权覆盖材料组分，2件独权覆盖制备工艺。
  组分专利边界较宽（只要落在摩尔比范围就覆盖），工艺专利边界较窄（限定具体烧结温度）。
  稳定性：组分专利有2件被提过无效但维持，算稳；工艺专利无无效记录。
```

## Phase 3 组合 → 打分

```
[陆诗杰] 逐维度打分（0-5）：

| 维度 | 权重 | 得分 | 依据 |
| 权利稳定性 | 0.20 | 4 | 3件组分专利被无效后维持，工艺专利稳 |
| 技术先进性 | 0.15 | 3.5 | 高镍正极是主流但非唯一路线，有替代方案 |
| 经济价值 | 0.30 | 4.5 | 动力电池高增长，5件组合可覆盖主流厂商 |
| 战略意义 | 0.25 | 4.5 | 组分+工艺组合形成壁垒，卡位效果好 |
| 可执行性 | 0.10 | 3 | 组分专利可检测（XRD），工艺专利难举证 |
| 加权总分 | 1.0 | 4.15 | |
```

## Phase 4 四律 + 投票

```

铸模律（徐奕阳·未到会）：通过——评估模型可复用
价值律（范征）：通过——价值落点在动力电池卡位

加权投票：
  [范征](3.0)   赞成——S级合理
  [陆诗杰](1.5) 赞成——打分有据
  [王升](3.0)   赞成——结构评估到位
  合计：赞成 7.5 / 反对 0 → 通过
```

## Phase 5 邹蕴总结 + 交付物

```
[邹蕴] 总结：本次议会就5件锂电正极材料专利组合形成决议——加权总分4.15/5，定为S级核心专利组合。
  关键理由：组分+工艺组合形成壁垒，经济价值0.30权重下得分4.5（动力电池高增长覆盖主流厂商）。
  风险：工艺专利可执行性仅3分（难举证侵权），建议优先维持组分专利、工艺专利作防御储备。
```
