# 算鱼圣人名册（22 位）· 路由数据源

> 本文件是 `/sptler` 的路由依据。每条记录含：身份 · 三国映射 · 官职 · 部门 · 专长 · 议会席位 · **路由关键词** · 代表语录。
> 路由算法：扫描用户问题，命中某圣人的「路由关键词」越多、专长越匹配，则其入会优先级越高。四核心按其「律」的关键词判定。

## 议会角色总览

- **议长/主持人（固定）**：邹蕴（决策圣）——权重 0，不投票，负责召集、控议程、四律检查、僵局仲裁、收口。
- **四核心（高权重投票）**：王升、张鑫、徐奕阳、范征 ——权重 **3.0**。
- **三大专业委员会首席（中权重投票）**：孙高德（平台）、蔡悦（跨界集成）、黄嵩泉（组合工程）——权重 **2.0**。
- **专科圣人（基础权重投票）**：其余圣人按议题路由入会 ——权重 **1.0**。

## 加权投票权重表

> 每位入会圣人都有投票权，但权重不同。最终计票 = 各人「赞成权重和」vs「反对权重和」（弃权不计）。议长邹蕴权重 0 不投票，仅在权重和接近（差距 ≤ 1.0）或平票时裁决。

| 权重 | 角色 | 圣人 |
|---|---|---|
| 0 | 议长（不投票） | 邹蕴 |
| 3.0 | 四核心 | 王升、张鑫、徐奕阳、范征 |
| 2.0 | 委员会首席 | 孙高德、蔡悦、黄嵩泉 |
| 1.0 | 专科圣人 | 喻学兵、卢若雨、顾峻峰、骆希聪、徐伟、钱文宇、陈哲锋、陆一帆、金辰宇、徐骋、陈方移、陈彤、江漪、陆诗杰 |

> **动态加成**：议题高度契合某圣人专长时，议长邹蕴可宣布该圣人本次权重 +0.5（动态投票）。每个议题至多 2 人获加成。
